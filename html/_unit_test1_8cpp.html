Функция для проверки наследования в конкретном классе.В ней написаны две функции , которые передаются по наследованию как public для дочерних классов (car,ship,motocycle) Каждая функция _start() и _finish() возвращает значение, которое чисто указывает номер дочернего класса для car(1) ship(2) и т.д. и т.п.

Parameters

Returns
Возвращает сумму номеров наших этих классов умноженных на 2.
